# 推奨環境

- node LTS 10.15.3
- npm 6.4.1 以上

##  開発サーバー立ち上げ

```
npx gulp

or

npm run gulp

or

yarn gulp
```

---

## 本番ビルド

```
npm run build

or

yarn build
```

---

## gulp コマンド一覧

```
npx gulp --tasks
```

├── build
├── clean
├── copy
├── default
├── pug
├── sass
├── server
├── reload
└── watch
